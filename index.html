How Strong is your Selector?

Specificity is key!

#unique{background-color:teal;}
.common{background-color:red;}
div{background-color:blue;}
div p strong{background-color:light-green;}

You can create strong selectors by chaining them together.

Cascading

p { color:pink;}

p { color:green;}

p { color:yellow;}

Order matters

Cascading

Strength increases as you travel down

CSS3/HTML5

section+p = all <p>s immediately after <section>

p:first-child = first <p> within it's parent

p:nth-child(even) = every other <p> starting with the second element.

p:first-of-type = the first <p> on the page

p:first-letter = selects the first letter within all <p>s

p:before/p:after = create pseudo elements within your <p>s

p[data-awesome='true'] = selects each <p> with the attribute 'data-awesome' set to true

media queries = not really selectors but target specific browsers/devices/orientations
